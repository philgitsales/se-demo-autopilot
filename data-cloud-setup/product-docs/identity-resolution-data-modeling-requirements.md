DATA 360
Unify Source Profiles

Use identity resolution in Data 360 to consolidate data from different sources into comprehensive views of your customers and accounts. Identity resolution uses matching and reconciliation rules to link data about people or accounts into unified profiles. Identity resolution creates complete, actionable views of your customers but doesn’t create golden records and isn’t a master data management (MDM) system. Each unified profile contains all the unique contact point values from all sources. You choose which view of your customers to use based on each business use case you want to solve.

About Identity Resolution
Identity resolution is the processing engine that generates unified profiles from source profile data stored in Data 360. Unified profiles act as a system of reference to unlock our source data. Understand how and when source profiles are processed and what you can do with unified profile output.
Profile Unification Types
Identity resolution in Data 360 offers several different ways to unify your records. The unification type you select determines what types of objects you can base your unified profiles on, the output objects created by the ruleset, and the speed at which records can be unified.
Identity Resolution Rulesets
Rulesets contain match and reconciliation rules that tell the customer data platform how to link multiple sources of data into a unified profile. Unified profile information is stored in data model objects created by the ruleset.
